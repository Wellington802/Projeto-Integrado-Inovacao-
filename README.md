## Projeto Integrado Inovação - Clínica Vida+

Este repositório contém o projeto desenvolvido para a disciplina de **Análise e Desenvolvimento de Sistemas** da Anhanguera - Águas Claras.

## Objetivo
Criar um sistema de controle e atendimento de pacientes para a Clínica Vida+, aplicando conceitos de:
- Programação em Python
- Estruturas de dados (listas, dicionários, funções)
- Expressões lógicas e tabelas verdade
- Pseudocódigo
- Diagramas de Casos de Uso

## Estrutura
- `src/` → Código em Python
- `docs/` → Relatório, diagramas e quadro Trello
- `pseudocodigo/` → Algoritmo de fila de pacientes
- `tabelas_verdade/` → Expressões lógicas e tabelas verdade

## Funcionalidades do sistema
1. Cadastrar paciente  
2. Ver estatísticas  
3. Buscar paciente pelo nome  
4. Listar todos os pacientes  
5. Sair
