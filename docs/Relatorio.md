# Projeto Integrado Inovação

Este documento contém o relatório do projeto desenvolvido para a disciplina de Análise e Desenvolvimento de Sistemas da Anhanguera - Águas Claras.

## Introdução
O objetivo foi desenvolver um sistema de controle e atendimento de pacientes para a Clínica Vida+, utilizando conceitos de lógica computacional e programação em Python. Foram aplicadas listas, dicionários, funções, condicionais, laços de repetição e validações de entrada. Também foram criadas tabelas verdade e diagramas de casos de uso.

## Desenvolvimento
- Menu interativo em Python com opções de cadastro, estatísticas, busca e listagem.
- Funções com validações usando `.strip()`, `.isalpha()`, `.isdigit()`, `.lower()`.
- Estrutura de dados: lista de pacientes e dicionário para cada cadastro.
- Tabelas verdade para consultas normais e emergenciais.
- Algoritmo em pseudocódigo para simular fila de atendimento.
- Diagrama de Casos de Uso para secretária, paciente e médico.

## Conclusão
O projeto consolidou o aprendizado técnico e mostrou como a programação pode ser aplicada para resolver problemas reais de forma eficiente e estruturada.
